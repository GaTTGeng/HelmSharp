# Code of Conduct

HelmSharp aims to be a practical and respectful open source project.

Expected behavior:

- Be respectful and constructive.
- Focus technical disagreement on facts, tradeoffs, and reproducible behavior.
- Avoid harassment, personal attacks, discriminatory language, or disruptive conduct.
- Respect maintainer time by keeping issues and pull requests clear and actionable.

Maintainers may edit, close, or remove comments, issues, pull requests, or other contributions that violate these expectations. Repeated or severe violations may result in a ban from project spaces.
